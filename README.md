# 🛍 Marketplace Card Bot

Telegram-бот на Python + aiogram 3 + SQLite + LLM API.  
Принимает фото или описание товара и генерирует полную карточку для маркетплейса.

## Что умеет

- Принять фото или текстовое описание товара
- Задать до 3 уточняющих вопросов при нехватке данных
- Сгенерировать карточку с:
  - Названием и описанием
  - УТП и преимуществами
  - SEO-ключами
  - Текстами для слайдов
  - Рекомендациями по фото
